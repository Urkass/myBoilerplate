# My personal boilerplate for react apps

First of all

```bash
npm i
```

then

### Development

```bash
npm start
```

Open http://localhost:8200/example/

### Components development

For executing storybook:

```bash
npm run storybook
```

Open http://localhost:9001/

### Production

```bash
npm run prod
```

Open http://localhost:9002/example/
